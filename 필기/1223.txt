**파이썬

-venv: Python의 표준 라이브러리
       별도의 패키지나 모듈환경을 제공하는 가상환경을 만들기 위해 사용

장점
다양한 프로젝트에서 서로 다른 패키지나 Python 버전의 충돌없이 작업가능
프로젝트별 필요한 패키지와 그 버전을 쉽게 관리
각 프로젝트의 의존성을 명확하게 확인가능

단점
각 가상환경마다 디스크 공간이 추가로 필요
여러 가상환경 사용 시 관리가 복잡할 수 있음



-주피터노트북: 오픈소스 웹기반 플랫폼
	       코드를 작성하기 쉽도록 환경을 조성
                   코드 자동완성이나 문법 체크 등의 편의기능 제공

장점
코드를 블록 단위로 실행해 인터프리터 언어인 파이썬의 특징 극대화
구글에서 제공하는 구글 코랩과 인터페이스 유사
사용이 편리한 인터페이스 제공

단점
시스템 표준 입출력 사용 불가
파이썬의 확장자와 주피터 노트북 파일의 확장자 상이해 변환작업 필요
